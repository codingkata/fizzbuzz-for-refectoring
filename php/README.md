## 运行说明

环境要求：PHP5或PHP7，且将PHP加入环境变量

+ #### Windows
三种方式选一种即可
```
1：      ./FizzBuzz.bat
2：      双击FizzBuzz运行
3：      chcp 65001
         php FizzBuzz.php
```
+ #### Linux
```
1：      php FizzBuzz.php
2：      chmod +x FizzBuzz.sh
         ./FizzBuzz.sh
```
+ #### Mac
```
1：      php FizzBuzz.php
2：      chmod +x FizzBuzz.sh
         ./FizzBuzz.sh
```
